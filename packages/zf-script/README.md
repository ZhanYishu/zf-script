# zf-script

个人快速获取模板工具命令

## 下载

```
npm install zf-script -g
```

## 配置
在根文件夹下配置个人模板地址
```
templatesRepo: 'https://github.com/ZhanYishu/templates.git'
```
具体模板结构参考：[https://github.com/ZhanYishu/templates.git]
## 使用

```
zf init <template-name> <project-name>
```
template-name: 仓库下模板文件名
project-name: 创建文件名称
